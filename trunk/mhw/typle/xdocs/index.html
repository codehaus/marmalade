Introduction

Typle is a Java class library that provides:

  • A data model for data types from various type systems (currently Java, MIME, etc.).
  • Mechanisms for reading type systems from persistent storage (currently XML).
  • Mechanisms for writing type systems to persistent storage in various formats (currently JavaBeans).
  • Ways to represent mappings from one type system to another.

Typle also provides tools that use the library to implement useful functionality, such as:

  • Converting simple record types into equivalent JavaBeans.
